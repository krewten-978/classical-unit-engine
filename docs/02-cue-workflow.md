# The CUE Workflow

This is the standard end-to-end workflow. A CUE run may stop after any stage if the teacher wants only that deliverable.

## Stage 0: Intake

Goal: define the assignment without overplanning.

Gather or infer:

1. central question or topic
2. grade level: 10th, 12th, or both
3. required authors/texts, if any
4. unit length: one handout, one week, multi-week, or paper sequence
5. final student product
6. source constraints: public domain, provided text, excerpts only, modern sources allowed, etc.

If enough information is provided, proceed. Do not ask unnecessary questions.

Deliverable: short intake summary.

## Stage 1: Inquiry framing

Turn the topic into an arguable central question.

Good central questions are:

- debatable
- intellectually serious
- answerable from texts
- not merely factual
- broad enough to connect readings
- narrow enough to support student writing

Deliverable: unit brief draft.

## Stage 2: Source discovery

Find candidate texts and passages. If the teacher gives only a topic, discover at least six serious candidates before asking for additions.

For each candidate, create a source card:

- author
- work
- passage
- source/citation
- key claim
- difficulty
- vocabulary load
- best grade level
- pairing possibilities
- role in unit

Deliverable: source discovery packet.

## Stage 3: Source selection and reading sequence

Choose the minimum strong set of readings. The sequence should create development or disagreement.

Possible sequence patterns:

1. ancient foundation → early modern challenge → modern complication
2. literary case → philosophical explanation → student judgment
3. authority position → rebellion position → prudential middle
4. fate/determinism → agency/responsibility → compatibility attempt
5. heroic ideal → tragic failure → moral judgment

Deliverable: recommended reading sequence with reasons.

## Stage 4: Difficulty judgment

For each selected passage, decide the conversion mode.

Modes:

1. faithful excerpt with support
2. lightly modernized excerpt
3. fully modernized prose
4. parallel original and paraphrase
5. teacher-provided excerpt left unchanged except for vocabulary/context

Deliverable: difficulty notes and conversion plan.

## Stage 5: Student-facing handout production

Build handouts in this order:

1. student-facing title
2. vocabulary preview
3. brief introduction
4. reading text
5. reading focus task

The reading focus should prepare students for argument, not merely check recall.

Deliverable: one or more student handout drafts.

## Stage 6: Argument mapping

For each major reading, identify:

- central claim
- supporting reasons
- assumptions
- key evidence
- what the view explains well
- what it struggles to explain
- strongest opponent
- possible student thesis

Deliverable: argument map and comparison map.

## Stage 7: Student production design

Choose work that students will produce along the way:

- window notes
- one-sentence claims
- paragraph responses
- dialogues between authors
- argument maps
- comparison paragraphs
- seminar claims
- paper thesis drafts

Deliverable: student task sequence.

## Stage 8: Paper prompt packet

Create an argument prompt requiring:

- clear thesis
- textual evidence from at least two readings
- fair treatment of an opposing view
- response to the opposing view
- explanation of why the student’s position best answers the central question

Deliverable: paper prompt packet.

## Stage 9: Differentiation

Create 10th and/or 12th grade variants as needed.

10th grade usually gets:

- shorter readings
- more vocabulary
- more context
- clearer section breaks
- guided claim work

12th grade usually gets:

- longer readings
- more original language
- more comparison
- stronger counterargument expectations
- fuller paper prompts

Deliverable: differentiated plan or separate differentiated materials.

## Stage 10: Packaging and verification

Before calling a CUE unit ready, check:

- Is the central question arguable?
- Are sources cited?
- Does the sequence create real disagreement or development?
- Are student handouts student-facing only?
- Is vocabulary useful but not bloated?
- Do tasks produce student work?
- Does the paper prompt require argument, evidence, and counterargument?
- Are 10th/12th versions differentiated by access rather than substance?

Deliverable: ready-to-teach packet.

## Stage 11: After-teaching revision

After the unit is taught, record:

- which readings worked
- where students struggled
- which vocabulary was necessary
- which tasks produced good thinking
- which prompts were too broad or too narrow
- what to cut next time
- what to add next time

Deliverable: notes_after_teaching.md.
