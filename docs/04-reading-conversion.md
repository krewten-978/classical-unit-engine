# Reading Conversion Standard

CUE converts difficult readings only after judging the actual source. The goal is access without distortion.

## Step 1: Judge the source

Ask:

- Is the problem vocabulary, syntax, context, argument structure, or all four?
- Can students handle the original with a vocabulary preview?
- Would light modernization be enough?
- Would a parallel original/paraphrase be better?
- Does the passage need to be shortened before it is adapted?

## Conversion modes

### 1. Faithful excerpt with support

Use when the source is close enough to grade level. Keep wording mostly intact. Add vocabulary, brief context, and a reading focus.

### 2. Lightly modernized excerpt

Use when the text has manageable ideas but outdated phrasing. Update syntax and archaic words while preserving the author’s sequence.

### 3. Fully modernized prose

Use when syntax or idiom blocks access. Preserve the argument, examples, and order of thought. Do not compress the passage into a mere summary.

### 4. Parallel original and paraphrase

Use for important but dense passages. Put the original excerpt next to a faithful paraphrase or below it in sections.

### 5. Unchanged teacher-provided excerpt

Use when Wayne supplies the exact text and wants support only.

## Required student-facing handout elements

1. title
2. subtitle or central question
3. vocabulary preview
4. brief introduction
5. reading
6. reading focus task

## Adaptation note

If a reading is modernized or adapted, say so plainly in the student-facing source note.

Examples:

- “Adapted into modern English from Hobbes, Leviathan, chapter 21.”
- “Lightly modernized from a public-domain translation.”
- “Excerpted from Book III with spelling and punctuation modernized.”

## Do not

- remove the author’s tension
- make every view sound moderate
- replace argument with a summary paragraph
- include teacher-facing notes in student handouts
- over-explain before students read
