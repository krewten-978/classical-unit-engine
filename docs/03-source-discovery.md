# Source Discovery Standard

CUE source discovery is not link collection. It is a disciplined pass through possible texts to decide what deserves classroom time.

## Discovery rules

1. Prefer stable, citable sources.
2. Prefer passages with clear claims.
3. Prefer disagreement over repetition.
4. Prefer teachable excerpts over famous but unusable chapters.
5. Preserve citation details.
6. Mark copyright or access limits when relevant.
7. Do not overstuff the unit.

## Preferred source types

For older texts:

- Project Gutenberg
- Perseus Digital Library
- Internet Classics Archive
- Fordham Sourcebooks
- Liberty Fund
- Wikisource
- university-hosted translations
- public domain editions

For modern sources:

- abstracts
- author-provided excerpts
- publisher previews
- encyclopedia summaries
- public lectures/transcripts where permitted
- teacher-provided PDFs or excerpts

## Source card scoring

Each source should be judged on five questions:

1. Does it answer or complicate the central question?
2. Does it make a claim students can identify?
3. Can it be excerpted without destroying the argument?
4. Is the difficulty worth the payoff?
5. Does it pair well with another source?

## Recommended source roles

Use one of these roles in the source card:

- foundation
- contrast
- complication
- literary case
- historical case
- theological frame
- political frame
- moral frame
- modern counterpoint
- optional extension

## Minimum first pass

When starting from a topic, the first discovery pass should normally include at least six serious candidates. The final unit may use fewer.

## Balanced historical topics

For contested historical questions, CUE should not rely only on one side’s self-justification or only on atrocity accounts. The source set should let students see rival claims, motives, fears, and moral judgments.
