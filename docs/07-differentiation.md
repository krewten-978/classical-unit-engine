# Differentiation Standard

CUE differentiates access, not substance.

## Shared intellectual core

Before differentiating, define what both versions must preserve:

- central question
- major authors/texts
- core conflict
- key terms
- final student product

## 10th grade version

Usually use:

- shorter excerpts
- more vocabulary support
- more context before reading
- clearer section headings
- one major comparison at a time
- window notes or paragraph claims before full essays
- guided thesis practice

Avoid:

- childish tone
- replacing argument with moral slogans
- too many authors in one packet
- removing the difficult idea that makes the text worth reading

## 12th grade version

Usually use:

- longer excerpts
- more original language
- less scaffolding inside the reading body
- more direct comparison among thinkers
- stronger counterargument requirements
- fuller paper prompts

Avoid:

- needless obscurity
- assuming context students have not been taught
- letting difficult prose substitute for difficult thinking

## Differentiation questions

For each reading, ask:

1. What must all students understand?
2. What vocabulary blocks access?
3. What context is necessary before reading?
4. What can be shortened without damaging the argument?
5. What writing task best prepares this grade level for the final product?
