# Run CUE Prompt

Use this prompt to ask an AI assistant to operate the workflow without building beyond the requested stage.

```text
Run CUE on the following topic/text. Follow the CUE repository workflow and stop at the stage I name.

Topic/text:

Stop after stage:

Grade level:

Final student product:

Constraints:

Do not create worksheets. Student tasks must produce claims, summaries, drawings/window notes, maps, dialogues, paragraphs, or argument papers. Preserve serious ideas. Differentiate access, not substance.
```

## Common stop points

- Stage 1: central question and unit brief only
- Stage 2: source discovery packet only
- Stage 3: recommended reading sequence only
- Stage 5: first handout draft
- Stage 6: argument map
- Stage 8: paper prompt packet
- Stage 10: complete ready-to-teach packet
